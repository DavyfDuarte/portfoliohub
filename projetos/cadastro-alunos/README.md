# Sistema de Cadastro de Alunos

Projeto desenvolvido na disciplina de Logica de Programacao — CEUB 2026.

## Descricao

Sistema de cadastro de alunos via terminal desenvolvido em Python.
Permite cadastrar, listar, buscar e remover alunos, alem de calcular
a media e estatisticas gerais da turma.

## Funcionalidades

- Cadastrar aluno com nome, curso e nota
- Aprovacao automatica (nota >= 5 = Aprovado)
- Listar todos os alunos em formato de tabela
- Buscar aluno pelo nome
- Remover aluno do sistema
- Calcular media da turma e contar aprovados/reprovados
- Validacao de entradas

## Conceitos aplicados

- Funcoes (def)
- Listas e dicionarios
- Estruturas de repeticao (for, while)
- Condicionais (if/elif/else)
- Tratamento de erros
- Modulo __main__

## Como executar
## Exemplo de uso
Escolha uma opcao: 1

Digite o nome do aluno: Davy

Digite o curso: Ciencia da Computacao

Digite a nota do aluno (0 a 10): 8.5
Aluno 'Davy' cadastrado com sucesso! Situacao: Aprovado
