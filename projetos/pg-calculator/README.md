# Calculadora de Progressao Geometrica

Projeto desenvolvido na disciplina de Matematica para Computacao — CEUB 2026.

## Descricao

Programa em Python que permite ao usuario calcular termos, soma e razao
de uma Progressao Geometrica (PG) por meio de um menu interativo no terminal.

## Funcionalidades

- Calcular os N primeiros termos de uma PG
- Calcular a soma dos N primeiros termos
- Descobrir a razao entre dois termos consecutivos
- Validacao de entradas do usuario
- Menu interativo com loop de repeticao

## Conceitos aplicados

- Funcoes (def)
- Estruturas de repeticao (for, while)
- Tratamento de excecoes (try/except)
- Logica matematica de PG

## Como executar
## Exemplo de uso

Escolha uma opcao: 1

Digite o primeiro termo (a1): 2

Digite a razao (q): 3

Quantos termos deseja calcular? 5
Termos da PG: [2, 6, 18, 54, 162]
